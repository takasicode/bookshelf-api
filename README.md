# Bookshelf API
Submission: Bookshelf API pada kelas Belajar Membuat Aplikasi Back-End untuk Pemula

## Run Locally
Clone the project
```bash
  git clone https://github.com/takasicode/bookshelf-api
```

Go to the project directory
```bash
  cd bookshelf-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
